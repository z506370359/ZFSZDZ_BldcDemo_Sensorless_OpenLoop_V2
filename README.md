# ZFSZDZ 无感方波开环控制项目 (Sensorless Open-Loop V2)

## 📌 项目概述

本项目基于张飞实战电子“扬帆起航”课程的核心理念开发，实现了一种无传感器（Sensorless）的 BLDC 方波开环速度控制方案。该方案通过反电动势（BEMF）过零点检测来实现换相，非常适合应用于对体积和成本有严格要求的显卡散热风扇等高速电机驱动控制场景。

## ✨ 核心特性

* **升频升压强拖启动**：按照升频升压（V/F）的启动流程，平滑实现电机的强拖运行。
* **闭环过零点检测**：在升频升压过程中引入了闭环过零点检测思路，实现精准的反馈换相。
* **平滑切入无感运行**：电机能够从启动阶段完美过渡，正常切入无感状态下的速度开环运行。

## ⚙️ 硬件与电机配置参数

项目的主控芯片底层依赖于 `stm32g030xx.h` 库，适用于 STM32G030 系列微控制器。以下是电机运行的核心宏定义参数配置：

### 1. 基础运行参数

* **默认旋转方向**：逆时针（CCW） `DEFAULT_DIR (DIR_CCW)`。
* **占空比限制**：
* 最大占空比（DUTY_MAX）：25% (设定值 500)。
* 最小占空比（DUTY_MIN）：15% (设定值 300)。
* 占空比步进（DUTY_STEP）：1。



### 2. 预定位（Alignment）参数

为了确保启动成功率，启动前会对转子进行预定位：

* **预定位占空比**：5% ~ 10% 左右 `ORIENT_DUTY (200)`。
* **预定位时间**：约 250ms `ORIENT_TIME (4000)`。

### 3. 强拖与反电动势（BEMF）参数

* **强拖换相时间范围**：
* 最长换相时间（对应约 300 RPM）：`FORCEPHASETIME_MAX (266)`。
* 最短换相时间（对应约 600 RPM）：`FORCEPHASETIME_MIN (133)`。
* 换相时间步进：10 `FORCEPHASETIME_STEP`。


* **反电动势过零系数**：理论值为 50%，代码中设定为 `BEMF_ZERO_FACTOR (30000)`，可根据实际电机运行情况灵活调整。
* **升频升压换相系数**：`PHASE_N_FACTOR (5242)`，用于 Vbus*N% 计算。

## 🧠 软件控制状态机

代码的控制逻辑由一个严密的状态机（State Machine）驱动，定义于 `Bldc.h` 中，包含以下核心状态：

1. **STOP_STATE (0)**: 停机状态。
2. **ORIENT_INITFIRST_STATE (1)** / **ORIENT_INITSECOND_STATE (2)** / **ORIENT_END_STATE (3)**: 预定位初始化及结束状态，用于转子初始位置对齐。
3. **FORCE_RUN_STATE (4)**: 强拖运行状态，执行升频升压控制。
4. **SENSORLESS_RUN_STATE (5)**: 无感运行状态，依赖反电动势过零点进行稳定换相。
5. **TEST_STATE (6)**: 预留的测试状态。
